What It Does

Three Fixes. Zero Effort.

Connect your Cloudflare account and GitHub repo — let Fix-it handle the rest.

🔒

SSL Auto-Renew

Detects expiring certificates and auto-renews via Cloudflare Origin CA — before they become outages.

  • Monitors certificate expiry daily
  • Triggers renewal at 14 days before expiry
  • Uses Cloudflare API — no DNS verification needed
  • Stores per-customer tokens in KV
  • Works with your Cloudflare account or EdgeIQ's
🛡️

Security Header Auto-Fix

Detects missing security headers and deploys a Cloudflare Worker to add them — CSP, HSTS, X-Frame-Options, and more.

  • One-click "Fix All Headers" deployment
  • Auto-deploys Cloudflare Worker to your account
  • Sets Strict-Transport-Security, CSP, and 5 others
  • Configurable CSP for your app's needs
  • Removes headers on demand
🐙

GitHub Issue Auto-Create

When a scan finds a vulnerability, automatically creates a formatted GitHub issue with severity, endpoint, and remediation steps.

  • Title format: "[EdgeIQ] Vulnerability: {type} on {domain}"
  • Rich body: severity, endpoint, description, remediation
  • Labels: "security", "edgeiq", plus priority label
  • Uses customer's stored GitHub PAT
  • Works with any public or private repo
Setup in Minutes

How It Works

Connect once. Fix-it handles everything after that.

1

Connect Cloudflare

Store your Cloudflare API token and Zone ID. We store it securely in per-customer KV.

2

Link GitHub (Optional)

Add your GitHub PAT so Fix-it can create issues in your repo when vulnerabilities are found.

3

Enable Fix-it

Toggle on the fixes you want. Choose manual trigger, automatic on detection, or scheduled.

4

Relax

Fix-it watches your infrastructure 24/7. Fixes happen automatically. You get notified either way.
