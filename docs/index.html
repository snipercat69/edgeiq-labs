How to Run a Free Web Scan

Method 2 — Terminal (Python 3)

All EdgeIQ tools run directly from your terminal with Python 3. No OpenClaw, no package manager required — just download and run. Works on Windows, macOS, and Linux.

Prerequisites

  • Python 3.8+ — python.org or via your system package manager
  • pip — usually included with Python
  • Optional API keys — some tools work fully free; others support premium API keys for higher limits

Quick Start — Download and Run

Each tool is a standalone Python script. Download the script for the tool you want, then run it directly:

XSS Scanner

Scans web applications for Cross-Site Scripting vulnerabilities with 40+ WAF bypass payloads.

# Download and run directly
curl -sL https://edgeiqlabs.com/dl/edgeiq-xss-scanner.py -o edgeiq-xss-scanner.py
python3 edgeiq-xss-scanner.py --target https://example.com/search?q=test

# Or with pip install (if available)
pip install edgeiq-xss-scanner

Network Scanner

Deep port scanning, service version detection, CVE matching, and OS fingerprinting.

# Download and run
curl -sL https://edgeiqlabs.com/dl/edgeiq-network-scanner.py -o edgeiq-network-scanner.py
python3 edgeiq-network-scanner.py --target example.com --ports 1-1000

# Full port scan
python3 edgeiq-network-scanner.py --target example.com --full

SSL / Security Headers Checker

Checks SSL certificate expiry, security headers, and WHOIS registration status.

# Download and run
curl -sL https://edgeiqlabs.com/dl/edgeiq-ssl-checker.py -o edgeiq-ssl-checker.py
python3 edgeiq-ssl-checker.py --domain example.com

Subdomain Hunter

Certificate Transparency scan, subdomain enumeration, and takeover detection.

# Download and run
curl -sL https://edgeiqlabs.com/dl/edgeiq-subdomain-hunter.py -o edgeiq-subdomain-hunter.py
python3 edgeiq-subdomain-hunter.py --domain example.com

Dark Web Credential Checker

Check if emails or domains appear in known data breaches.

# Download and run
curl -sL https://edgeiqlabs.com/dl/edgeiq-credential-checker.py -o edgeiq-credential-checker.py
python3 edgeiq-credential-checker.py --email your@email.com

Getting API Keys (Optional)

All tools work in free mode. Adding an API key increases rate limits and unlocks premium features:

# Set your API key as an environment variable
export EDGEIQ_API_KEY=your_api_key_here

# Or pass it directly as an argument
python3 edgeiq-xss-scanner.py --target https://example.com --api-key your_api_key

Python Dependency Installation

For tools that require extra Python packages, install dependencies with pip:

# Install all EdgeIQ tools as a package
pip install edgeiq-tools

# Or install individual tools
pip install edgeiq-xss-scanner edgeiq-ssl-watcher edgeiq-subdomain-hunter

Add to PATH (Optional)

Move tools to a folder in your PATH so you can run them from anywhere:

# On Linux/macOS — add to ~/.local/bin
mv edgeiq-xss-scanner.py ~/.local/bin/edgeiq-xss
chmod +x ~/.local/bin/edgeiq-xss
edgeiq-xss --target https://example.com

# On Windows — add to User PATH
# Download the .py file, then run:
python3 edgeiq-xss-scanner.py --target https://example.com

Automation with Cron

Schedule scans to run automatically on Linux/macOS:

# Schedule a daily SSL check at 9am
(crontab -l 2>/dev/null; echo "0 9 * * * /usr/bin/python3 /path/to/edgeiq-ssl-checker.py --domain example.com --quiet") | crontab -

# Schedule a weekly subdomain scan
(crontab -l 2>/dev/null; echo "0 8 * * 1 /usr/bin/python3 /path/to/edgeiq-subdomain-hunter.py --domain example.com") | crontab -

Windows Task Scheduler (Alternative to Cron)

# Create a scheduled task on Windows to run daily at 9am
schtasks /create /tn "EdgeIQ Daily SSL Check" /tr "python3 C:\tools\edgeiq-ssl-checker.py --domain example.com" /sc daily /st 09:00

# Run it now
schtasks /run /tn "EdgeIQ Daily SSL Check"

Setting Up Alerts

Get notified when scans find issues — via email, Slack, Discord, or Telegram:

# Install the alerting system
pip install edgeiq-alerting-system

# Configure a Discord webhook
export DISCORD_WEBHOOK=https://discord.com/api/webhooks/your-webhook-here

# Test alerting
python3 -m edgeiq_alerting --test

# Run a scan with alert on critical findings
